Introduction:

This tool takes a peak table file as input and performs total intensity normalization within samples. The formula is (x/sum of total intensity within the corresponding sample) *1000.

Input files:

1.     Peak table file in Tab-delimited text format with the first column as the compound identifier and others as samples.
